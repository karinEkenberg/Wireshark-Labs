# Network Protocol Analysis Lab
This repository contains a structured lab environment for analyzing network traffic using Wireshark. The goal is to understand protocol behavior, identify anomalies, and practice traffic analysis for security purposes.

## Lab Objectives
* Analyze the structure and headers of key network protocols.
* Identify patterns of normal vs. suspicious traffic.
* Document findings to build a personal knowledge base for incident response.

## Directory Structure
Each directory is named after a specific protocol and contains:
* Packet capture files (.pcap/.pcapng).
* A local README.md with detailed analysis and findings.

## Methodology
1. Capture traffic or use sample datasets.
2. Filter for specific protocols.
3. Inspect handshakes, payloads, and termination sequences.
4. Document the "Golden Standard" (normal behavior) to easily spot deviations.
